EMA web map

The Erasmus Mundus Association is a professional network for Erasmus Mundus students and graduates. This webmap serves as a search tool for graduate programs by location and links prospective students to program webpages.

There are two modes: Program and City. The first mode allows one to filter the programs in mapview by program name and displays markers for all locations of an individual program. The second displays programs and links to individual webpages by city. Discontinued programs are marked D/C. For more details, please see the list of all Erasmus Mundus Masters Courses.

We hope that this program exploration tool will serve as a resource for students, faculty, and EMA staff.
